Pong

an age-old computer classic, redesigned to be infuriating as can be!

built using PoseNet, so that you will always struggle to get it to identify your wrist

Instructions

You control the left paddle.

Move your right wrist up and down to control the paddle.

You get a point for hitting the ball, but the computer gets one if you fail to.

Four points for the computer ends the game and you lose.

Winning is impossible.
